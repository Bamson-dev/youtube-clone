A clean, fast YouTube-inspired video platform built with HTML, CSS, and JavaScript.
This project replicates the core YouTube UI with less clutter, better spacing, and a faster feel.

## Pages

- Home page with video grid feed
- Video watch page with embedded player and recommendations
- Search results page
- Channel profile page

## Features

- Dark mode by default
- Available On Desktop alone (might fix the mobile if i am asked to .. lol)
- Clean navigation with collapsible sidebar
- Video cards with thumbnail, title, channel, views, and upload date
- Filter tabs for content categories
- No ads, no autoplay previews, no visual noise

## Tech Stack

- HTML
- CSS
- JavaScript
